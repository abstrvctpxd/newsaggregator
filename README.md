# newsaggregator
Web dashboard that fetches headlines from multiple news providers (e.g., NewsAPI, Guardian API) and aggregates them into a single feed.
Key Concepts: Merging data from multiple endpoints, filtering by keywords/categories, handling pagination, and deduplicating articles. 
Why it matters: Teaches you to orchestrate multiple APIs simultaneously and normalize different JSON structures into a unified format. 
Extension: Add a feature to save interesting articles to a local SQLite database or send a daily digest email.
